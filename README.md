# Test Technique - Vaybe

Application Fullstack de gestion de candidatures réalisée en 4h.

## Stack Utilisée

- **Frontend :** React.js (Vite), CSS3
- **Backend :** Node.js, Express
- **Logique :** Système de scoring automatique intégré

## Installation

1. Cloner le projet : `git clone https://github.com/GodsonDGB/vaybe-test.git`
2. **Backend :** - `cd backend`
   - `npm install`
   - `node server.js` (Lancement sur le port 5000)
3. **Frontend :**
   - `cd frontend`
   - `npm install`
   - `npm run dev`
